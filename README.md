****Installation Guide****
**1. Install Node.js**

  Download and install Node.js from:

  https://nodejs.org
  
  After installation, verify:
  
  node -v
  npm -v
  
**2. Install Vite**

  Install Vite globally:
  
  npm install -g vite
  
  Verify installation:
  
  vite --version
  
**3. Install Project Dependencies**
  npm install
  
**4. Install Three.js**
  npm install three
  
**5. Run the Development Server**
  npm run dev

Open the Local URL in your browser.
